Folders inside MAP_PROJECT_FILES:-

1. MAP_PROJECT.rar :- Includes all files to be run on Android Studio. Copy this file in your PC where inside Android Studio Projects folder and ruin the project.

2. MAP_PROJECT.apk :- This is the apk file to be installed in android phone and run the application.

3. SYNOPSIS:- This is the summary of the application.

4. CODES.pdf :- This is the PDF file which includes code and the screenshots from the application.
